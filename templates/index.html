This webapp is used to interact with ethereum blockchain via web3





Sr Operation Description Link
1 Block lookups Blocks can be fetched by providing either the block number or hash using API also can get the latest block. Note: Block hash should be in their hexadecimal representation. Block Lookups
2 Check Balance find the amount of ether owned by an account. Provide with ether public address. Check Balance
3 Making transactions send ether from one account to another. sign transaction and broadcast transaction via other provider. Interact with contracts and transaction lookups. Transaction operations
4 Interact with contracts Interact with existing contracts, deploy new contracts. Contract operations
5 Test applications Get accounts, balance, transaction and send transactions Testing. Transaction operations

This application is solely made for learning purposes. The web provider is Http and the main net node is connected to infura. To get the sample block accounts we have used eth-tester. Interacting with contracts is currently in development.
